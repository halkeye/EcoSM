Admin commands:
/chat -- Shows Commands related to chat.
/districts -- Shows commands related to user-defined districts.
/initialspawn -- Controls initial spawn.
/inventory -- List of inventory related commands
/meteor -- Shows commands for manipulations with meteor.
/network -- List of different network commands
/notifications -- Shows commands related to notifications.
/performance -- Runs server performance reports and dump to files. Optionally pass duration in seconds, defaults to 60 seconds.
/profiler -- Shows commands to profile the server and generate diagnostic data.
/teleport -- List of different teleportation commands
/weather -- Shows commands for manipulations weather.

User commands:
/civics -- Performs a variety of government operations.
/help (/?) -- Displays all the commands available with hidden subcommands. Accepts a string to filter commands to a search string.
/helpful -- Shows all help, including sub commands.
/land -- List of commands for land manipulations
/manage -- Shows commands for users administration.
/money -- Shows commands for manipulating money.
/sim -- List of commands for simulations
/time -- List of commands for time operating
/util -- List of some helping commands
